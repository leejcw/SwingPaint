SwingPaint
==========
An Interative 2D Constructive Solid Geometry (CSG) Demo
-------------------------------------------------------
![CSG Bear](/images/csg-bear.png =400x)

### Operators
`a` : AND (cyan)

`o` : OR (green)

`n` : NOT (red)

`x` : XOR (yellow)

### Shapes
`r` : rectangle

`s` : square

`q` : quadrilateral

`e` : ellipse

`c` : circle

`t` : triangle

### Controls
`del` : remove selected

`tab` : cycle selected

arrow keys: expand/shrink shape

space : cycle corners
